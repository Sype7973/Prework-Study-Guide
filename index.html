Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML code can be seen as the framework of the structure you are trying to build.

CSS

  • CSS is how the information will form on the page and how it cascades
  • CSS is also used to present your data/information and how it is designed
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is the complete solution to starting, building and completing a coding project
  • Git can be used to collaborate with other developers too complete projects
  • Git has a lot of features to ensure your development process adheres too your style of developing for example; Agile or Waterfall
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Javascript is utilised to create interactable information on your website/app
